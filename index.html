Cómo leer este mapa

Las líneas unen el orden sugerido (izquierda → derecha; luego baja a la siguiente fase). Las tarjetas con “Paso” son hitos del recorrido; las que dicen “Función del portal” son pantallas del mismo entorno después de entrar: útiles, pero no cuentan como paso nuevo del alta. Cada tarjeta abre la pantalla real del demo. A la derecha de cada fase, integraciones típicas (pasa el cursor o enfoca un chip para ver el detalle).

No se cargó la configuración. Verifica modules.config.js.
